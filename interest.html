INTEREST – A TO Z (SI & CI)

All banking exam scenarios • Formulas • Approach • Solved logic
1. SIMPLE INTEREST – CORE
Simple InterestSI = (P × R × T) / 100
AmountA = P + SI
Approach:
Use SI when interest is calculated only on principal (linear growth).
Example 1:
P = ₹4000, R = 5%, T = 3 years
SI = 4000×5×3/100 = ₹600
2. WHEN RATE (R) IS NOT GIVEN
From CI − SI (2 yrs)CI−SI = P(R/100)²
Approach:
If difference between CI and SI for 2 years is given → rate is hidden inside square.
Example 2:
P = ₹2500, CI−SI = ₹25 (2 years)
25 = 2500(R/100)² → R² = 1 → R = 10%
3. WHEN TIME (T) IS NOT GIVEN
Amount RatioA / P = (1+R/100)ᵀ
Approach:
Convert problem into base ratio, then identify power.
Example 3:
Amount becomes double at 10% CI.
2 = (1.1)ᵀ → T ≈ 7 years
4. CI – CORE
Compound AmountA = P(1+R/100)ᵀ
Compound InterestCI = A − P
Example 4:
P = 1000, R = 10%, T = 2
A = 1000×1.21 = 1210 → CI = 210
5. CI − SI SHORTCUT (VERY IMPORTANT)
For 2 yearsCI − SI = P(R/100)²
Example 5:
P = 5000, R = 8%
Difference = 5000×(8/100)² = ₹32
6. FRACTIONAL YEARS
2½ years(1+R/100)² × (1+R/200)
Example 6:
P = 2000, R = 10%, T = 2½
A = 2000×1.21×1.05
7. HALF / QUARTERLY COMPOUNDING
Half-yearly(1+R/200)²ᵀ
Quarterly(1+R/400)⁴ᵀ
Example 7:
10% p.a. half-yearly → 5% for 2 periods/year
8. SUCCESSIVE INCREASE / DECREASE
Net % changex + y + xy/100
Example 8:
Increase 20%, then 10%
Net = 20+10+2 = 32%
9. SI = CI CONDITION
Rule:
SI = CI only for 1 year (always).
Example 9:
At any rate, for 1 year → SI = CI
10. FIND TIME WHEN SI = PRINCIPAL
TimeT = 100 / R
Example 10:
R = 8% → T = 12.5 years
BANKING GOLD SUMMARY
✔ If CI−SI given → think square of rate
✔ If double/triple amount → use ratio power
✔ Fraction years → split year
✔ Half/quarter → rate ÷ periods, time × periods
✔ Never blindly apply formula – identify missing value first
